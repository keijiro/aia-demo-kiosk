The size animation settings for the particle system in the ShockwaveFX prefab are as follows.

Settings Overview

This particle system is configured so that the size linearly expands from 0 to 1.2 over 0.18 seconds, from the moment the particle is born until it disappears. This creates the visual effect of a shockwave spreading out rapidly.

Detailed Parameters

  1. Start Size: 1.2
    • This is the base size that the particle ultimately reaches.
  2. Start Lifetime: 0.18 seconds
    • The time from when a single particle is created until it disappears. The size animation completes within this short period.
  3. Size over Lifetime: Enabled
    • Curve setting: Linear increase from 0 (at birth) to 1 (at death).
    • Formula: Start Size (1.2) × Curve value (0–1)
    • Behavior: The particle starts at size 0 (invisible) at the moment it is created, then grows uniformly to size 1.2 over 0.18 seconds.
  4. Separate Axes: Disabled
    • The particle expands uniformly across all X, Y, and Z axes (isotropic expansion).

Visual Effect

Because the particle expands from 0 to 1.2 in an extremely short time (0.18 seconds), it is perceived by the player as a "sharp, instantaneous burst of impact." Features such as Size by Speed are not used; this is a simple expansion animation controlled purely by the passage of time.
