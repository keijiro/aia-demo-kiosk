1. ゲーム概要

  • ジャンル: 2Dトップダウンの「サバイバー系（Survivors-like）」アクションゲーム。
  • コアコンセプト: 「反射（Reflect）」。プレイヤーは魔法使いですが、直接魔法を撃つのではなく、敵の弾を「結界（バリア）」で跳ね返して攻撃します。
  • メインループ:
    1. 移動: マウスカーソルを追従して移動。
    2. 戦闘: 画面外から迫る敵が放つ大量の弾を、タイミングよく結界（左クリック）で反射。反射した弾は速度が倍増し、敵にダメージを与えます。
    3. 成長: 敵を倒してXPを獲得し、レベルアップ時に「移動速度」「クールダウン」「結界の大きさ」などの強化を選択。

2. ビジュアルとスタイル

  • アートスタイル: 64-bitピクセルアート、古典的なファンタジー（剣と魔法）の世界観。
  • 描画ルール:
    • キャラクターは常に「斜め前向き（南西方向）」の歩行アニメーションを再生。
    • 右方向に移動する際は、スプライトを左右反転して対応。
    • 8色のカラーパレット（Kino.Eight）による統一感のあるビジュアル。

3. 技術的仕様・開発ガイドライン

  • 入力システム: New Input Systemを使用。ただし、Action Mapは使用せず、Keyboard.current や Mouse.current を使った直接的な入力検知を行う。
  • UIシステム:
    • メインメニューやレベルアップ画面には UI Toolkit (UITK) を使用。
    • ゲーム内のインジケーター（クールダウンバー等）には UGUI/ワールド空間スプライト を使用。
  • アーキテクチャ: PlayerController を情報の中心（Source of Truth）とするコンポーネントベース。
  • 注意点: レベルアップ時は Time.timeScale = 0 で停止するため、UIアニメーション等は unscaledTime で制御する必要があります。

このプロジェクトにおいて、新しいキャラクターの追加や機能の実装が必要な場合は、これらのガイドライン（特に描画の向きや入力の実装方法）に沿って進めることができます。

    private void OnEnable()
    {
        var root = GetComponent<UIDocument>().rootVisualElement;
        _startLabel = root.Q<Label>("startLabel");
        _fadeOverlay = root.Q<VisualElement>("fadeOverlay");

        // Start blinking effect
        StartCoroutine(BlinkRoutine());
    }

    private void Update()
    {
        if (_isStarting) return;

        // Using New Input System directly as per Guidance.txt
        bool mouseClicked = Mouse.current != null && Mouse.current.leftButton.wasPressedThisFrame;
        bool keyPressed = Keyboard.current != null && Keyboard.current.anyKey.wasPressedThisFrame;

        if (mouseClicked || keyPressed)
        {
            StartGame();
        }
    }

    private void StartGame()
    {
        _isStarting = true;
        StopAllCoroutines();
        
        // Ensure label is visible when clicked
        _startLabel.style.opacity = 1f;
        
        StartCoroutine(FadeAndStartRoutine());
    }

    private IEnumerator BlinkRoutine()
    {
        while (!_isStarting)
        {
            _startLabel.style.opacity = 1f;
            yield return new WaitForSeconds(0.3f);
            _startLabel.style.opacity = 0f;
            yield return new WaitForSeconds(0.2f);
        }
    }

    private IEnumerator FadeAndStartRoutine()
    {
        float duration = 0.5f;
        float elapsed = 0f;

        // Reset fade overlay opacity just in case
_fadeOverlay.style.opacity = 0f;

        while (elapsed < duration)
        {
            elapsed += Time.deltaTime;
            float t = elapsed / duration;
            _fadeOverlay.style.opacity = t;
            yield return null;
        }

        _fadeOverlay.style.opacity = 1f;
        
        // Load the Main scene (Index 1)
        SceneManager.LoadScene(1);
    }
}
